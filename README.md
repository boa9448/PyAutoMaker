PyAutoMaker
========
자동화를 위한 파이썬 패키지

개발중인 버전이라 내보낸 API는 언제든지 변경될 수 있습니다


설치방법
========

[PyPI package](https://pypi.python.org/pypi/PyAutoMaker/)에서 설치하는 방법

    pip install PyAutoMaker

또는 저장소에서 클론하는 방법

    git clone https://github.com/boa9448/PyAutoMaker
    cd PyAutoMaker
    python setup.py install
